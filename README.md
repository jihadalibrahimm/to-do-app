## About
A simple **To-Do List** app to manage your daily tasks. You can add, edit, delete, mark as completed, search, and hide completed tasks. Also includes a "mark all as completed" toggle.

---

## Features
- Add new tasks  
- Edit tasks via a modal  
- Delete tasks  
- Toggle task completion  
- Delete all completed tasks  
- Mark all tasks as completed (toggle)  
- Search tasks  
- Hide completed tasks  

---

## Tech Stack
- React (with `useState`)  
- JavaScript ES6+  
- HTML5 & CSS3  
- Vite
